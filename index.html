Project

Crime Rate Prediction Using Machine Learning

This is a powerful project which can detect future crime rates, rapes, and murders in each state. It’s designed to indicate where the focus of improvement in India should be. The project uses an ML model to predict crimes, murders, and rapes in the future based on factors like literacy rates, unemployment rates, poverty rates, and police per 100,000 population. The ML model combines these factors, weighs their importance for each state, and derives predictions with a certain level of accuracy.

This enables law enforcement agencies and policymakers to make informed decisions.
